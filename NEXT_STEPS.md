## Option 2: Deploy to Production

### Backend Deployment

#### Option A: Heroku
```bash
# Install Heroku CLI
# Create Procfile
# Deploy
heroku create rosepay-api
git push heroku main
```

#### Option B: AWS/DigitalOcean
- Use PostgreSQL instead of SQLite
- Set up environment variables
- Configure domain & SSL

#### Option C: Railway/Render
- Easy deployment
- Automatic SSL
- Database included

### Frontend Deployment

#### Option A: Vercel (Recommended)
```bash
cd frontend
npm run build
vercel deploy
```

#### Option B: Netlify
- Connect GitHub repo
- Auto-deploy on push
- Free SSL

#### Option C: GitHub Pages
- Static hosting
- Free for public repos
